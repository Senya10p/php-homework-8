1. В папке classes создали класс DB (в файле DB.php), класс новостей News (файл News.php), класс статьи Article (файл Article.php).
Также находится файл View.php с классом View из предыдущего ДЗ.
Параметры соединения берем из файла config.php.

2. Создаём таблицу news.
Для этого создаём из консоли новую датабазу hw8. 'CREATE DATABASE hw8'
Затем подключаемся к датабазе hw8 через php storm.
Настройки подключения находятся в файле config.php.

Для создания таблицы используем запросы:
CREATE TABLE news (id SERIAL, header VARCHAR(100), text TEXT, author VARCHAR(100));

3. Создаём две страницы сайта, используем технику шаблонов и класс View.
